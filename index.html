Rogue

Rogue is a basic rogue like written in C. I chose to use C, as you can use it on any platform essentially. The game is currently a work in progress.

Build

To build, compile main.c with your system's C compiler (gcc, clang, cc, whatever) If you're on macOS, you can use the handy build.sh script.

Game Details

This game can run on basically any system with a keyboard and a terminal. It's only been tested on macOS, but it doesn't use any external libraries so it shouldn't be a issue.
